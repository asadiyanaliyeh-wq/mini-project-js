# پروژه مدال (Modal) 

یک پروژه ساده HTML، CSS و JavaScript برای نمایش **پنجره مدال (Modal)** 

- باز شدن با کلیک روی دکمه
- بستن با کلیک روی × یا خارج از مدال
- بستن با کلید Escape
---

## تکنولوژی‌ها

- HTML
- CSS (با تم دارک و فونت Google Fonts)
- JavaScript (برای باز و بسته کردن مدال)
- Font Awesome (برای آیکون ×)
