# 🌸 تولیدکننده QR Code

یک پروژه ساده، کاربردی و زیبا برای ساخت **QR Code** با استفاده از  
**HTML، CSS و JavaScript**

این برنامه به شما اجازه می‌دهد با وارد کردن یک متن یا آدرس اینترنتی،
به‌صورت فوری QR Code مربوط به آن را تولید کنید.

---

## ✨ امکانات

- ✅ تولید QR Code از متن یا لینک
- 🎨 رابط کاربری صورتی و مینیمال
- ⚡ عملکرد سریع و بدون نیاز به بک‌اند
- 🌐 استفاده از API رایگان QR Code
- 📱 قابل اجرا در تمام مرورگرها

---

## 🚀 نحوه استفاده

1. پروژه را اجرا کنید (باز کردن فایل `index.html`)
2. یک متن یا آدرس اینترنتی وارد کنید
3. روی دکمه **«تولید QR Code»** کلیک کنید
4. QR Code شما نمایش داده می‌شود
